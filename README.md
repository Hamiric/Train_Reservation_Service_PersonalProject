#### < 필수 기능 >
>* UI 명세서를 정확하게 반영 : 
<br>UI 명세서에 적힌대로 레이아웃 작성<br><br>
>* HomePage 구현 : 
<br>출발역, 도착역을 선택할 수 있는 초기 화면 구현<br><br>
>* StationListPage 구현 : 
<br>기차역 리스트를 보여주고 선택할 수 있는 화면 구현<br><br>
>* SeatPage 구현 :
<br>좌석을 선택하는 화면 구현
#### < 도전 기능 >
